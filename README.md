## Requisitos

* PHP 8.2 ou superior
* Composer
  
## Como rodar o projeto baixado

## Sequencia para criar o projeto
Criar o projeto com laravel
```
composer create-project laravel/laravel .

Iniciar o projeto criado com laravel
```
php artisan serve
```
Acessar o conteúdo padrão do Laravel
```
http://127.0.0.1:8000
